## Project Overview

This repository was generated by UIGen and contains a single-file React portfolio component (`App.jsx`) for Sujan Khadka. There is no build system configured yet — the component is intended to be integrated into a host application that provides React, Tailwind CSS, and lucide-react.

## Architecture

**`App.jsx`** — the entire application is one default-exported React component (`Portfolio`) with:
- State: `isMenuOpen` (mobile nav), `scrollY` (parallax), `activeTab` (project filter)
- Inline data arrays for `projects`, skills, and testimonials
- Tailwind CSS for all styling (dark mode via `dark:` variants)
- lucide-react for icons
- Profile image imported from `@/images/Sujan_Pic.js` as a Base64 data URL

**`images/Sujan_Pic.js`** — exports a single Base64-encoded image string; the `@/images/` alias must be configured in the bundler.

## Setup (not yet configured)

The repo has no `package.json`. To use this component you need a host project with:
- React 18+
- Tailwind CSS (with dark mode class strategy)
- lucide-react
- Path alias `@` → project root (e.g., in Vite: `resolve.alias: { '@': '.' }`)

Typical commands once configured:
```
npm install
npm run dev    # dev server
npm run build  # production build
```

## Key Patterns

- Project filtering works by matching `project.category` against `activeTab`; add new categories to the filter button list and ensure `category` values match exactly.
- Parallax effects use `scrollY` state and inline `transform: translateY(...)` styles.
- The mobile menu toggles a full-screen overlay; `isMenuOpen` drives both the hamburger icon swap and the overlay visibility.
